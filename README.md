# Password-Strength-Checker

A terminal-based Python program that checks the strength of a user-entered password based on common security rules and provides suggestions to improve it.


## Features
- Checks minimum password length
- Detects uppercase and lowercase letters
- Detects digits and special characters
- Assigns a strength level (Weak / Moderate / Strong)
- Provides suggestions to make the password stronger


## Technologies Used
- Python 3
- Terminal / Command Line


## How to Run
1. Make sure Python is installed on your system
2. Clone this repository or download the files
3. Run the program using:
python password_checker.py

## Sample Output
Enter your password: kumar78

Password Strength Analysis:
Strength : WEAK
Score    : 2 / 5

Suggestions to make it stronger:
- Increase password length to at least 8 characters
- Add at least one uppercase letter (A-Z)
- Add at least one special character (@ # $ % & ! *)
